Welcome to the CL Request Non-Static Values Page!

Search for a specific record in the table by typing in the search box above (under the CL Request Non-Static Value CLS header) and then clicking the magnifying glass icon.
Show all hidden values under the tag accordions by check-marking "Show All Records". To hide all values under the tag accordions, uncheck "Show All Records"
To show changes made to the table (in order of most recent changes), click the "View Changes" button (under the CL Request Non-Static Value CLS header) to view the list of changes made to the table so far.
Add Record:
Click on the "Add Request" button (under the CL Request Non-Static Value CLS header), fill out the pop-up box with the desired tags and values, and confirm adding the record by clicking "Add" in the pop-up box. After adding the request, the record should appear inside a green box in the list of records.
Delete Record:
Click on the trash can icon next to the request tag/value. After deleting the request, the record should appear inside a red box in the list of records.
Edit Value:
Click on the value's text in the list of records and type in the new value. After clicking off of the value's text (editing the request), the value should appear inside a yellow box in the list of records.
Once you are satisfied with the changes made, click the "Save Changes" button to permanently save the changes committed to the list of records (add, delete, and edit changes mentioned above)
